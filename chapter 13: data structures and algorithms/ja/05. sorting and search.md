# ソートと探索 (Sorting and Search)

- **比較ソート**: バブルソート、挿入ソート、マージソート、クイックソート (ピボット選択戦略、Hoare vs Lomuto パーティション)、ヒープソート
- **比較ソートの下限**: 決定木による計算量 O(n log n) の証明
- **非比較ソート**: 計数ソート (Counting sort)、基数ソート (Radix sort)、バケットソート
- **二分探索 (Binary search)**: 標準的な二分探索、下限・上限の探索 (lower/upper bound)、答えに対する二分探索 (単調関数の利用)
- **分割統治法**: マスター定理、マージソートの解析、最近接点対問題
- **強欲アルゴリズム (Greedy algorithms)**: 活動選択問題、ハフマン符号、区間スケジューリング
- **動的計画法 (DP)**: 部分問題の重複、最適部分構造、メモ化 vs タブレーション、古典的問題 (ナップサック問題、LCS: 最長共通部分系列、編集距離、コイン換え問題)
- **バックトラッキング**: N クイーン問題、数独、制約充足問題

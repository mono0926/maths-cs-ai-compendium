# クラウドコンピューティング

- **クラウドサービスモデル**: IaaS、PaaS、SaaS、FaaS (サーバーレス)
- **主要プロバイダーの概要**: AWS、GCP、Azure — コンピュート、ストレージ、ネットワークの基本要素
- **仮想化**: ハイパーバイザー (Type 1、Type 2)、VM vs コンテナ
- **コンテナ**: Docker (イメージ、レイヤー、Dockerfile)、コンテナレジストリ
- **オーケストレーション**: Kubernetes (Pods、Services、Deployments、StatefulSets、DaemonSets)、Helm
- **ストレージ**: ブロック (EBS)、オブジェクト (S3/GCS)、ファイル (EFS/NFS)、データレイク
- **クラウドにおけるネットワーク**: VPC、サブネット、セキュリティグループ、ロードバランサー、サービスメッシュ (Istio、Envoy)
- **サーバーレス**: Lambda/Cloud Functions、コールドスタート、イベントトリガー
- **コスト管理**: スポット/プリエンプティブルインスタンス、予約済み容量、オートスケーリングポリシー
- **Infrastructure as Code (IaC)**: Terraform、CloudFormation、Pulumi

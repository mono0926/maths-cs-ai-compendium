# 高度なテキスト生成

- **テキスト拡散モデル (Text diffusion models)**: D3PM、MDLM、連続テキスト拡散、離散データ向けのスコアマッチング
- **テキスト OCR**: 生活シーンにおけるテキスト検出・認識パイプライン（Scene text detection/recognition）、CRNN、ドキュメント OCR、Vision-Language OCR
- **制御可能な生成 (Controllable generation)**: テキスト向け Classifier-free guidance、RLHF（人間のフィードバックによる強化学習）、DPO、Constitutional AI
- **超長文文脈対応 (Long-context methods)**: スパース・アテンション、スライディングウィンドウ、リング・アテンション、メモリ拡張モデル
- **検索拡張生成 (Retrieval-Augmented Generation: RAG)**: Retriever-Reader アーキテクチャ、Dense Passage Retrieval (DPR)、チャンキング戦略
- **投機的デコーディング (Speculative decoding)**、Medusa heads、並列生成

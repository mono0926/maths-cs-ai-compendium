# テキスト処理と古典的 NLP

- **テキストの正規化 (Text normalisation)**: ケースフォールディング（大小文字の統合）、Unicode の取り扱い、正規表現、編集距離（レーベンシュタイン距離）
- **トークン化 (Tokenisation)**: 空白区切り、ルールベース、サブワード（BPE, WordPiece, Unigram, SentencePiece）
- **ステミング (Stemming) とレンマタイゼーション (Lemmatisation)**
- **品詞タグ付け (POS tagging)**: ルールベース、HMM タガー、ビタビアルゴリズム
- **固有表現抽出 (Named entity recognition: NER)**: 系列ラベル付け、BIO タグ
- **構文解析 (Syntactic parsing)**: CYK アルゴリズム、シフト還元パーサ、依存構造解析
- **Bag-of-words (BoW)**、**TF-IDF** 表現
- **N-gram 言語モデル**、パープレキシティ、スムージング（ラプラス、Kneser-Ney）

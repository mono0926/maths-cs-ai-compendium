# プログラミング言語 (Programming Languages)

- **言語パラダイム**: 命令型、オブジェクト指向、関数型、論理型
- **型システム**: 静的 vs 動的、強い型付け vs 弱い型付け、型推論、ジェネリクス
- **メモリ管理**: スタック vs ヒープ、手動管理 (C/C++)、ガベージコレクション (トレース型、参照カウント)、所有権モデル (Rust の借用チェッカー)
- **コンパイル**: 字句解析、構文解析 (AST: 抽象構文木)、意味解析、コード生成、LLVM
- **インタプリタと実行環境**: バイトコード VM (JVM、CPython)、JIT (実行時) コンパイル
- **主要な言語機能**: クロージャ、パターンマッチング、代数的データ型、トレイト/インターフェース
- **ドメイン固有言語 (DSL)**: SQL、正規表現、シェーダー言語
- **言語設計のトレードオフ**: パフォーマンス vs 安全性 vs 表現力

# x86 と AVX

- **x86 SIMD の進化**: MMX → SSE → SSE2/3/4 → AVX → AVX2 → AVX-512 → AMX
- **AVX/AVX2 プログラミング**: 256 ビット YMM レジスタ、イントリンジック (_mm256_\*)、FMA 命令
- **AVX-512**: 512 ビット ZMM レジスタ、マスクレジスタ、ギャザー/スキャッター (Gather/Scatter)、競合検出 (Conflict detection)
- **Intel AMX**: タイルレジスタ、TMUL (Tile Matrix Multiply)、BF16/INT8 加速
- **メモリアライメント**: アラインメント済み vs 未アラインメントのロード、キャッシュラインの考慮事項
- **パフォーマンスの落とし穴**: AVX による周波数の低下 (Throttling)、レジスタ・プレッシャー、レーンを跨ぐペナルティ
- **ベンチマークとプロファイリング**: RDTSC、perf、VTune、likwid

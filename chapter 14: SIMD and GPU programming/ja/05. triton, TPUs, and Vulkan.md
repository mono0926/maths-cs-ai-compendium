# Triton、TPU、Vulkan

- **Triton**: Python ベースの GPU カーネルプログラミング、ブロックレベルの抽象化、自動チューニング
- **Triton カーネルの記述**: tl.load、tl.store、tl.dot、マスキング、グリッド/ブロックプログラム
- **Triton vs CUDA**: 生産性と制御のトレードオフ、それぞれの使い分け
- **Flash Attention のケーススタディ**: タイリングによるメモリ効率の高い Attention、オンライン Softmax
- **TPU アーキテクチャ**: シストリックアレイ (Systolic arrays)、MXU (Matrix Multiply Unit)、HBM、ICI インターコネクト
- **TPU プログラミング**: XLA コンパイラ、JAX/pjit、GSPMD、シャーディング (Sharding) 注釈
- **Vulkan Compute**: コンピュートシェーダー、SPIR-V、デスクリプタセット、コマンドバッファ
- **比較**: GPU (CUDA/Triton) vs TPU (JAX/XLA) vs Vulkan、適切なツールの選択

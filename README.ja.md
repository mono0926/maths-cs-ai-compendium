## 概要 (Overview)

ほとんどの教科書は、優れたアイデアを難解な記法の中に埋もれさせ、直感的な説明を省き、読者がすでに内容の半分を知っていることを前提としています。そして、AI のような進歩の速い分野ではすぐに内容が古くなってしまいます。このコンペンディウム（大要）は、数学、コンピューティング、人工知能を基礎からカバーする、オープンで型破りな教科書です。単に試験や面接を乗り切るためではなく、これらを深く理解したいと願う知的好奇心旺盛な実践者のために書かれています。

## 背景 (Background)

AI/ML 分野で働いてきた数年間、私はノートを「直感的な理解、現実世界での文脈、そして数学・コンピューティング・AI 概念の（誤魔化しのない）説明」で埋め尽くしてきました。2025年、数名の友人がこれらのノートを使って DeepMind、OpenAI、NVIDIA などの面接準備を行いました。彼らは全員合格し、現在それぞれの役割で活躍しています。そこで、この内容をすべての人に共有することにしました。
